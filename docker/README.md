# Docker — Guía de tecnologías

Introducción a Docker para perfiles con conocimientos básicos de programación que quieren entender cómo funciona la contenedorización, cómo escribir un `Dockerfile` a mano y cómo un asistente de IA lo genera automáticamente.

| # | Archivo | Por qué leerlo aquí |
